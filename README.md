## Описание

**YUAI Server** — это высокопроизводительный многопротокольный сервер, поддерживающий **HTTP/1, HTTP/2, HTTP/3 (QUIC) и WebSocket**. Он включает систему черного списка IP-адресов и обработку очередей через **Redis Streams**.

## Возможности

- Поддержка **HTTP/1, HTTP/2 и WebSocket** (порт `443`).
- Поддержка **HTTP/3 (QUIC)** (порт `4433`).
- Хранение черного списка IP в **Redis**.
- Блокировка IP после **5 неудачных попыток авторизации**.
- Ограничение длины запросов (**не более 8 KB**).
- Обработка очереди запросов через **Redis Streams**.
- Защищенный доступ к API с авторизацией по **Basic Auth**.

## Установка и запуск

### 1. Клонирование репозитория

```sh
git clone https://github.com/cthvlab/yuaiserver.git
cd yuaiserver
```

### 2. Установка зависимостей

Убедитесь, что у вас установлен **Rust** и **Cargo**. Затем установите зависимости:

```sh
cargo build --release
```

### 3. Настройка Redis

Запустите **Redis** локально:

```sh
redis-server
```

### 4. Запуск сервера

```sh
cargo run --release
```

## API эндпоинты

### 1. Добавление IP в черный список

**Запрос:**

```http
POST /blacklist/add
Authorization: Basic <admin:password>
Content-Type: text/plain

192.168.1.100
```

**Ответ:**

```json
{"message": "IP 192.168.1.100 добавлен в очередь на блокировку"}
```

### 2. Доступ к корневой странице

**Запрос:**

```http
GET /
```

**Ответ:**

```text
Welcome to YUAI SERVER
```

## HTTP/3 (QUIC) поддержка

Для работы с HTTP/3 необходимо использовать **quiche** или **curl** с поддержкой QUIC:

```sh
curl --http3 https://localhost:4433
```
